- Java JDK 8 or higher
- Any Java IDE (Eclipse, IntelliJ, VS Code) or command line

## Installation & Usage

1. **Download the file**
   - Download `ProfessionalBiodataSystem.java`

2. **Compile and Run via command line:**
   - javac ProfessionalBiodataSystem.java
   - java ProfessionalBiodataSystem
  
3. **Or run in IDE:**
- Open the file in any Java IDE
- Run the main method

## How to Use

1. Fill Personal Information - Name, age, gender, and upload photo
2. Enter Academic Details - Registration, branch, semester, CGPA
3. Add Contact Information - Phone, email, social profiles, address
4. Select Skills & Hobbies - Check predefined options or add custom ones
5. Provide Additional Info - About me, achievements, and projects
6. Generate & Export - Preview, print, or save as text file

## Form Sections

- Personal Information - Basic details and photo
- Academic Information - Education and institution details
- Contact Information - Multiple contact methods and address
- Skills & Interests - Technical skills and hobbies with custom options
- Additional Information - Personal description and achievements

## Technical Details

- Framework: Java Swing
- UI Components: Custom-styled JComponents
- Validation: Regex patterns for email and phone validation
- File Handling: Text export and image upload support
- Printing: Java Print API integration
